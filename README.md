# Playwright Automation Framework (TypeScript)

## 📌 Project Overview

This project is a scalable end-to-end test automation framework built using Playwright with TypeScript. It follows the Page Object Model (POM) design pattern for better maintainability and readability.

## 🚀 Features

* Page Object Model (POM) architecture
* Data-driven testing using JSON
* End-to-End test coverage (Login, Cart, Checkout)
* Cross-browser testing (Chromium, Firefox, WebKit)
* Parallel execution and retry mechanism
* HTML test reports
* GitHub Actions CI/CD integration

## 🧪 Test Scenarios Covered

* User login (valid & invalid)
* Add product to cart
* Verify cart items
* Complete checkout process

## 🛠️ Tech Stack

* Playwright
* TypeScript
* Node.js

## ▶️ Run Tests

```bash
npx playwright test
```

## 📊 View Report

```bash
npx playwright show-report
```

## 🔄 CI/CD

Tests run automatically using GitHub Actions on every push.
